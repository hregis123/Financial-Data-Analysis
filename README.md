# Financial Data Analysis & Profit Optimization for a Lending Institution

## Project Overview
This project was conducted as a freelance Data Analysis mission for a French financial institution specializing in credit lending. The primary objective was to evaluate the institution's portfolio performance for the year 2021, identify financial risks, and analyze profitability across different branches and loan types.

## Key Objectives
* **Risk Assessment:** Identify clients approaching their maximum repayment capacity (debt-to-income ratio).
* **Profitability Analysis:** Calculate net profits generated by the institution, segmented by city and branch.
* **Trend Visualization:** Analyze the monthly evolution of profits throughout 2021, broken down by loan category (Mortgage, Consumer loan, etc.).
* **Reporting:** Summarize findings into actionable financial insights.

## Technologies and Libraries
* **Language:** Python
* **Environment:** Jupyter Notebook / Google Colab
* **Libraries:**
    * `pandas`: Data cleaning, transformation, and feature engineering.
    * `matplotlib` / `seaborn`: Exploratory Data Analysis (EDA) and financial trend visualization.
    * `numpy`: Numerical calculations for interest rates and debt ratios.

## Analysis Workflow
1. **Data Preprocessing:** Handled missing values and standardized financial formats.
2. **Feature Engineering:** * Calculated the **Debt Ratio** for each client to assess credit risk.
    * Computed the **Monthly Profit** per loan using interest rates and principal amounts.
3. **Exploratory Data Analysis (EDA):**
    * Aggregated data by branch location to identify the most profitable regions.
    * Segmented profits by loan type to determine which products drive the most revenue.
4. **Data Visualization:** Created time-series plots to monitor profit fluctuations and seasonal trends during 2021.

## Key Results
* **Risk Mitigation:** Isolated a specific segment of high-risk clients based on debt-to-income thresholds.
* **Strategic Insights:** Identified the top-performing branches and highlighted the loan types with the highest return on investment (ROI).
* **Forecasting Basis:** Provided a clean, structured dataset ready for future predictive modeling.
